## 0.1.0 — Initial Release

### Features

- **Core editor** — ProseMirror-based document model with immutable state
- **Security layer** — DOMPurify sanitization on all HTML input, URL validation,
  CSS injection prevention, tab-napping protection
- **Extensions** — Bold, Italic, Underline, Strike, TextColor, Highlight, Link,
  Image, Paragraph, Heading (h1–h6), BulletList, OrderedList, ListItem,
  CodeBlock, Table, TableRow, TableCell, TableHeader, FontFamily
- **Google Fonts** — full bundled Google Fonts catalog with configurable
  picker subsets, lazy loading via FontFace API, validated names, and CDN allowlist
- **React UI** — Toolbar, BubbleMenu, FloatingMenu, FontPicker, LinkPopover,
  ColorPicker, TableControls
- **Themes** — Light, dark, and auto (system preference) via CSS custom properties
- **Next.js support** — SSR-safe with `"use client"` boundary, CSP helpers
- **Accessibility** — ARIA labels, keyboard navigation, focus management,
  reduced-motion, high-contrast mode support
- **TypeScript** — Full type coverage, branded types for security-sensitive values
